RAG 기반 로컬 문서 탐색기

사용자가 직접 제공한 학교/프로젝트 문서를 로컬에서 학습하고, 필요한 양식과 근거 문서를 채팅으로 찾는 RAG 시연용 페이지입니다. 개인정보가 포함된 파일은 다운로드 시 자동으로 마스킹본을 제공합니다.

앱 열기: localhost:8501 실행 방법 보기
문서 업로드 및 학습

저장소에는 기본 문서가 없으며, 시연자가 준비한 PDF, TXT, MD, CSV 파일만 로컬에서 학습합니다.

채팅형 검색

질문을 입력하면 본문 유사도, 파일명 점수, 사용자 피드백을 합쳐 관련 문서를 찾습니다.

마스킹 다운로드

이메일, 전화번호, 학번, 이름 등 민감정보가 감지되면 삭제한 파일을 제공합니다.

시연 흐름

1시연자가 준비한 예시 문서 업로드
2문서 학습 버튼 실행
3필요한 양식 질문
4근거 문서 확인 및 마스킹본 다운로드

예시 대화

업로드한 문서에서 필요한 양식 찾아줘
사용자가 제공한 로컬 문서 안에서 관련 문서 후보를 찾고, 근거 문서와 다운로드 옵션을 표시합니다.
본문 유사도 파일명 매칭 민감정보 마스킹

다운로드 안전 처리

안전한 파일은 원본 그대로 다운로드합니다. 민감정보 패턴이 감지된 파일은 가능한 경우 같은 형식의 마스킹본으로 제공합니다.

xlsx 마스킹본 docx 마스킹본 pptx 마스킹본 pdf 마스킹본 txt/csv/md 마스킹본

실행 방법

터미널에서 프로젝트 폴더로 이동한 뒤 아래 명령을 실행합니다.

.\venv\Scripts\python.exe -m streamlit run app_streamlit.py --server.port 8501

브라우저에서 다음 주소를 엽니다.
